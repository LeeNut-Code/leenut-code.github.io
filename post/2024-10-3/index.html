一些好用的安卓app推荐

发布于 2024年10月03日 | 标签: 技术 | 3分钟 | 651字数 | 浏览量::

本文将推荐一些好用的开源app和国外app

日历、待办

  • Etar：开源日历，ui很丑
    GitHub

  • Google日历：可通过Google 账号，同步和导入日程
    play商店

  • 一叶日历：酷安

  • 日历备份：用来导入和导出.ics日历文件，ics日历文件请在互联网搜索获取
    play商店

  • 微软To Do：应用宝

天气

  • Breezy Weather：开源天气app，数据来自国外平台
    GitHub

笔记

  • fairnote：支持webdav同步
    果核剥壳

  • SilentNotes：支持android/PC，可用多种同步方式
    果核剥壳

  • Notally：好像暂不支持webdav
    GitHub

闹钟

  • 使命闹钟：部分品牌手机振动功能失灵
    官网

文件

  • 流舟文件：支持ftp、webdav
    酷安

  • 质感文件：GitHub

相册

  • Aves：GitHub

课表

  • wake up课程表：官网

翻译

  • papago：官网
  • DeepL：官网
  • 微软翻译：官网

软件管理

  • My APK：果核剥壳

播放器

  • MX播放器：官网
  • 绿乐：酷安

网盘

  • onedrive：国内无法打开web页面，pc/android客户端可以打开，免费存储5GB
    官网

  • Koofr：网站和app无中文界面，支持连结Google drive和OneDrive，免费存储10GB
    官网

  • 坚果云：支持webdav，有上传流量限制
    官网

  • 阿里云盘：存储容量很大，有一定的限速，隐私安全无保障（不建议上传关乎个人隐私的文件）
    官网

  • Google Drive：国内无法打开web页面/app，免费存储15GB，隐私安全可保障。
    官网

邮箱

  • outlook：官网
  • Gmail：官网

文档处理、图书阅览

  • Markor：用来写markdown的app。
    GitHub

  • Acode：手机上写代码。
    GitHub

  • onlyoffice：官网

  • 阅读：开源阅读
    官网

网络代理工具

  • clash：Apkpure

  • Shadowrocket：提供免费节点。
    Apkpure

新闻

  • 联合早报：第三只眼看中国
    官网

下载器

  • ADM：支持16线程下载。
    小众软件

即时通讯

  • Delta Chat：加密的电子邮箱IM，建议用来在多个设备间传输文件。
    官网

浏览器

  • via：小而美的浏览器，依靠webview，可定制性高，支持油猴脚本。
    官网

  • firefox：官网

  • edge：官网

社交媒体

  • bilimiao：开源第三方哔哩哔哩客户端。
    GitHub

输入法

  • Gboard：谷歌键盘。
    火哥分享

百科

  • 维基百科：GitHub

设备管理 相关

  • 冰箱：用来冻结系统内置的app.
    官网

  • 安装狮：安装app，可以用来绕过某些手机的app备案检测安装app.
    GitHub

  • 权限狗：管理app权限，保护用户隐私安全。
    官网

  • 小黑屋：冻结系统内置的app.
    官网

自律 & 锁机

  • 不做手机控：酷安
  • 远离手机：酷安
