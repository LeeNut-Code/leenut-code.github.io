Cloudflare Workers AI 使用教程：调用与工具集成实践

发布于 2026-04-15 | 标签: AI 、 Cloudflare | 4分钟 | 882字数

Cloudflare Workers AI 是 Cloudflare 推出的无服务器边缘 AI 推理平台。它允许开发者通过简单的 API 调用，在全球 300+ 个 Cloudflare 边缘节点上运行主流开源 AI 模型，无需管理任何 GPU 基础设施。

简单来说：你不需要买显卡、不需要部署模型、不需要考虑扩容——只需要一个 API 密钥，就能在你的应用里集成大语言模型、图像识别、代码生成等 AI 能力。

特点 说明
按量计费 采用“神经元（Neuron）”作为计算单位，用多少付多少

每日免费额度 支持 Meta、Google、Mistral、Qwen、DeepSeek、OpenAI 等多家主流模型

计费单位：神经元（Neuron）：Cloudflare Workers AI 不按传统的“请求次数”或“GPU 时长”计费，而是引入了一个抽象的计算单位——神经元（Neuron）。

支持的任务类型：文本生成（Text Generation）、文本嵌入（Text Embeddings）、文本分类（Text Classification）、文本转图像（Text-to-lmage）、文本转语音（Text-to-Speech）、自动语义识别（Automatic Speech Recognition）、图像转文本（Image-to-Text）、图像分类（Image Classification）、翻译（Translation）等。

如何通过 API 调用 Workers AI

要调用 Workers AI，你需要两个关键信息：Account ID 和 API Token。
在开始一切之前，请确保有一个Cloudflare账号。

REST API 获取

  1. 登录你的账号，进入 Workers AI页面（https://dash.cloudflare.com/{你的账号}/ai/workers-ai/usage）
  2. 点击 REST API按钮，创建Workers AI API令牌，在弹出的右侧侧栏中点击创建API令牌。
  3. 将你的帐户 ID和API令牌复制（API Token 仅显示一次，关闭页面后无法再次查看
    ）。

在本地工具中调用

Workers AI 提供了 OpenAI 兼容的 API 端点,不支持其他协议，具体调用格式和参数如下：

  • API 地址： https://api.cloudflare.com/client/v4/accounts/{您的账户ID}/ai/v1
  • API 密钥：您的API令牌
  • 模型名称格式：@cf/组织名/模型名，具体可用模型可以在 Cloudflare Workers AI 模型文档
